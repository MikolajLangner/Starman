# Starman
Simple 1v1 platform game

The bitmaps are not mine and links to them are added to the file 'links.txt'

Controllers:

PLAYER1:

Up - 'W'
Down - 'S'
Left - 'A'
Right - 'D'
Shoot - 'Spacebar'

PLAYER2:

Up - 'Up'
Down - 'Down'
Left - 'Left'
Right - 'Right'
Shoot - 'Period'

Have fun and let me know if there is sth that could be fixed or/and improved.
